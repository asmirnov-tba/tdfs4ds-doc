    tdfs4ds.benchmark.report ¶

    tdfs4ds.benchmark.report¶

    Render a compelling, self-contained HTML report from the dict produced by :func:tdfs4ds.benchmark.run_benchmark (or its saved JSON).

    The HTML is fully offline — inline SVG charts + CSS, no external JS — so it renders identically in any browser and embeds directly in the admin panel.

    :func:build_benchmark_html, :func:write_benchmark_report. Also a CLI:

    python -m tdfs4ds.benchmark.report report.json [-o out.html][--open]
